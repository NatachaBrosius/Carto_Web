Les Objectifs

Proposer une solution pour :

Mise en garde

Ce travail est produit dans le cadre d'un projet scolaire de fin de programme de DEC en Géomatique au Cegep Limoilou.

Il n'a aucune valeure légale.

Les scores affichés sont la moyenne des évaluations faites par les utilisateurs du site ayant fréquenté l'école.

Ils ne représentent pas un taux de réussite académique.

Ils peuvent être fluctuants en fonction:

  • du programme
  • de l'année scolaire

Le score affiché au départ est une moyenne globale, il sera adapté en fonction des filtres (programme et/ou année) sélectionnés.
5 étoiles grises signifient qu'il n'y a pas d'évaluation pour cette école (pour la moyenne générale) ou pour le programme choisi (si l'école propose le programme choisi)

Pour qui?

Source

Les données sources qui permettront la réalisation sont téléchargées :

  • Via le site données ouvertes pour les écoles et les routes
  • Via le site de la commission scolaire pour la liste des écoles, les bassins disponibles qui permettront de comprendre le découpage des bassins
  • Via Statistique Canada pour les informations démographiques
